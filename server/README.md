# WEB ADMIN KARAOKE PLUS

## 1. Giới thiệu:
Xây dựng web admin cho app Karaoke giúp, quản trị viên quản lý người dùng, bài đăng, bài hát và xử lý báo cáo một cách nhanh chóng, tiện lợi để người dùng có được trải nghiệm tốt nhất.

## 2. Yêu cầu môi trường:
 Để chạy được dự án này, bạn bắt buộc phải có:
 Node.js phiên bản v22.0.x trở lên, npm phiên bản 10.8.x trở lên
 **Kiểm tra:**

        node -v
        npm -v

**Cài đặt dự án:**
#### 1. Clone repository:

        https://github.com/HPhi1808/Karaoke.git

#### 2. Vào folder server và tạo file .env:

        cd server
        copy .env.example .env
#### 3. Gán giá trị cho các Key trong file .env vừa tạo
#### 4. Cài đặt môi trường:

        npm install
#### 5. Chạy dự án:

        npm start
